## git具体使用

用户现在github上建立一个仓库springboot

git init 初始化仓库

git add . 添加

vim .git/config:在该文件中添加自己的工作区
[user]
name = github用户名
email = 自己邮箱

git remote add origin https://github.com/tinggirl00/springboot.git
git push -u origin master
