## git具体使用

用户现在github上建立一个仓库springboot

git init 初始化仓库

git add . ：将修改的内容放在缓存中

vim .git/config:在该文件中添加自己的工作区
[user]
name = github用户名
email = 自己邮箱

git remote add origin https://github.com/tinggirl00/springboot.git
git push -u origin master


多次提交的具体步骤：
git status
git add .
git status
git commit -m "备注"
(git pull)将远端的代码拉下来
git push
